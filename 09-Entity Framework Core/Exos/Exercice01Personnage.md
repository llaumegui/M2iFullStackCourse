# Exercice 01 Personnage

## Objectif
Apprendre à maitrîser Entity Framework

## Sujet
1. Créer une classe modèle nommée Personnage qui contient les propriétés suivantes :
	- Id
	- Pseudo
	- PointsDeVie
	- Armure
	- Degats
	- DateCreation
	- NombrePersonneTues
2. Créer l'IHM suivant :
	1. Créer un personnage
	2. Mettre à jour un personnage
	3. Afficher tous les personnages
	4. Taper un personnage
	5. Afficher les personnages ayant des PVs (PV + armure) supérieurs à la moyenne
3. Si un personnage a perdu toute son armure et sa vie, il doit être supprimé !
